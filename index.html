JobFunnel

JobFunnel is a tool for scraping job postings from multiple websites into a single .csv file. I built it to save time spent clicking through job websites and encountering duplicates, irrelevant results, and advertising.

It has since gained a global userbase with ~1.5k stars on Github and an active development community.

Python BeautifulSoup Requests Threading Scraping

COVID-Net

At DarwinAI, I have had the opportunity to work closely with Dr. Alexander Wong, and have assisted in the development of COVID-Net, which is a neural network that is capable of identifying the presence of pneumonic COVID-19 from a chest X-Ray.

This tool is able to discriminate well between other pneumonic infections and COVID-19. Additionally, in-conjunction with Gensynth's Inquire tool, insights can be obtained around the triggering information in the image.

I focused primarily on building transfer learning tools to demonstrate that a measure of severity can be obtained as well. A complete version of this is realized as Covid-Net-S, which can be found on the project's Github.

Tensorflow Python Jupyter Imaging Transfer-Learning Root Cause Analysis

Aterica Veta™

As a mechanical design contractor, I helped bring Aterica's smart epi-pen case to reality. Working from an initially 3D-printed prototype, I refined the design for injection moulding and built initial prototypes. I additionally spent development time analyzing failure-modes and designing components which accomodate the environmental conditions and material composition.

The Veta™ is intended to save lives by providing real-time location and temperature feedback to people who rely on epinephrine auto-injectors.

The Aterica Veta™ is now purchasable online.
